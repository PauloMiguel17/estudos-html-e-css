Trabalhando com Listas

Listas Ordenadas

listas ordenadas podem ser criadas usando a tag <ol> e dentro a tag <li>

  1. Acordar
  2. escovar os dentes
  3. tomar café
  4. ir para academia
  5. treinar
  6. chegar em casa
  7. estudar html e css

usando a palavra type dentro da tag <ol> podemos definir como vai ser a numeraçao, e o start para definir a partir de qual numero irá comecar
